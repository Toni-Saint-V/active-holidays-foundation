# Polish Checklist

## UI quality

- Есть ли ясная иерархия?
- Один ли главный CTA?
- Нет ли лишних карточек, границ, акцентов?
- Читается ли экран быстро на mobile?

## Motion quality

- Есть ли явный enter/exit для важных переходов?
- Показывает ли motion причинно-следственную связь?
- Есть ли reduced-motion fallback?
- Нет ли рваных stagger/blur/scale эффектов?

## Performance quality

- Не перегружен ли основной bundle?
- Нужно ли code-splitting на route level?
- Нет ли тяжёлых компонентов, которые можно lazy-load?
- Не рендерятся ли большие интерактивные блоки без нужды?

## 10/10 signals

- экран ощущается собранным
- нет визуального шума
- motion ощущается дорогим, но тихим
- интерфейс отвечает быстро
- нет компромисса с a11y

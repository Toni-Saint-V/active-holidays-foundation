# AI Interactivity Checklist

## AI-native value

Ask:
- Помогает ли экран принять решение быстрее?
- Снимает ли AI неопределённость?
- Делает ли AI продукт честнее и прозрачнее?
- Улучшает ли AI handoff к человеку, а не только “выглядит умно”?

## Strong AI patterns

- live verdict preview
- conflict explanation
- missing-data guidance
- confidence breakdown
- scenario simulation
- next-best action
- summary for human expert

## Strong interaction patterns

- reveal on demand
- compare alternatives
- direct parameter tuning
- preview before commit
- retry / recover flows
- contextual drill-down

## Anti-patterns

- chat widget ради галочки
- псевдо-AI copy без пользы
- перегруженные панели
- interaction overload on mobile
